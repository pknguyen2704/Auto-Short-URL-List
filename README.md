# Auto-Short-URL-List
A automation process to short a list of URL links

## Tạo môi trường ảo
python3 -m venv venv 

source venv/bin/activate

## Cài đặt các thư viện cần thiết
pip install -r requirements.txt

## Chuẩn bị dữ liệu
linksList.csv: Danh sách các link cần rút gọn

## Run code
python3 main.py
